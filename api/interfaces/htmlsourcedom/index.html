Interface: HTMLSourceDom

A source which content is a DOM tree created by the transient render engine parseDocument method.

See useAmbientTRenderEngine.

remarks When you use a DOM source, the onHTMLLoaded callback will never be invoked for this source, since the source loader hasn't access to the HTML source of the DOM.
